# Consortium Node Framework
### Sails, socket, mysql, ember, SASS

### Getting Started

- Install Node.JS ([nodejs.org](http://nodejs.org/))
- Install NPM ([npmjs.org](//www.npmjs.org/))
- Install Sails : sudo npm -g install sails
